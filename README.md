# Minecraft MLP Orchestrator (核心API)

---

## 專案簡介

Minecraft MLP Orchestrator 是一個專為 Minecraft MLP（Machine Learning Pipeline）的核心 API。
這個 API 是paper, velocity版本專案的上游，提供最基礎且可在Minecraft運行的MLP功能。

---

## 使用方式

### 1. 搭配 orchestrator-paper 或 orchestrator-velocity 使用

至[orchestrator-paper](https://github.com/kfcisme/orchestrator-paper.git) 或 [orchestrator-velocity](https://github.com/kfcisme/orchestrator-velocity.git) 
下載**jar**檔案即可使用預設MLP功能。

### 2. 作為依賴庫使用

如果想在自己的專案中使用 Minecraft MLP Orchestrator 的功能，可以將其作為依賴庫引入。
本專案提供最基本的MLP功能，聚集高擴充性。
